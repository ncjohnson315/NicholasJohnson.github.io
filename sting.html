Excel Data Analysis &
Tableau Dashboard
for The Icon: Sting

Analyzing the career of wrestling icon Sting in Excel in order to create an interactive Tableau dashboard.

Data Collection and Cleaning in Excel

Data was collected from various sources and copied into an Excel spreadsheet. From there, data required various cleaning procedures and transformations.

Date Formatting

Dates were formatted into yyyy-mm-dd format.

Time Duration Formatting

Time duration was formatted into hh:mm:ss format, and an additional column was calculated for total time duration in seconds.

Geographic Location Formatting

Geographic location was split from one column into three columns (city, state/province, country). Some data was also in German, which had to be filtered and translated.

Separation and Distinction of Opponents and Teammates

Various wording was used to distinguish opponents from teammates in the original data. Therefore, several transformations were used with different delimiters, left/right functions, etc. to split the information into two columns (opponents and teammates).

Other Cleaning Procedures

Data was trimmed of whitespace, spelling errors were corrected, missing data was found (when possible), and capitalization of data was corrected.

Data Analysis in Excel

Data was analyzed using pivot tables to determine which information would be best suited for the dashboard.

  • Total matches by promotion
  • Total matches by state/province
  • Total matches by event type
  • Total matches by opponent
  • Average time duration of matches per opponent

Tableau Dashboard

Data was then imported into Tableau for the creation of the dashboard.

Table Representing Win-Loss Records by Promotions

Table was created using the Result column grouped by the Promotion column, with Count of Total Matches text added, filtering out any null result values, and sorting by decreasing Count of Total Matches.

Map Representing Total Matches by State/Province

Map was created using the State/Province column, with the Count of Total Matches color added, and adjusting the color legend due to much higher counts in Florida and Georgia than any other geographic area.

Stacked Bar Chart Representing Total Matches by Year by Event Type

Stacked bar chart was created using Count of Total Matches row, with Year column, Event Type color added, and adjusted the Event Type legend manually so that more important match types were on top.

Butterfly Chart Representing Total Matches by Opponent and Average Match Time Duration

Butterfly chart was created using Total Matches and Time Duration columns, Opponent row, filtering Opponent by top 10 by Count of Total Matches, adding a context filter by Promotion, and sorting by decreasing Count of Total Matches.

There were several null values in the time duration column which impacted the butterfly chart. This was fixed by creating a calculated field using the code below:

IF ZN(AVG([Duration Sec]))=0
THEN 0.0001
ELSE ZN(AVG([Duration Sec]))
END

Then a Constant Reference Line was utilized to keep the butterfly chart from misformatting itself with this small data point in place of the null. It was made transparent and utilized to keep the chart labels in place when null data was present.

Creating the Dashboard

The Dashboard was then created with these sheets, utilizing the Promotion filter from the Butterfly Chart to filter all sheets within the dashboard. A graphic of Sting's autograph was added near the title for visual effect.
